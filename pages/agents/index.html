Agent 应用教学
总览

本系列从“能落地”的视角，系统讲解智能体（Agent）的核心能力与工程化路径：如何将 LLM 与工具、知识库、状态管理、规划与协作结合，构建可靠、可评测、可迭代的实际应用。

面向任务
从需求出发，拆解为可验证的步骤与工具链。
工程化
状态、记忆、规划、容错、观测与评测闭环。
安全与合规
提示注入、工具滥用、权限边界与审计。
学习路径建议
  • 快速上手：从零到一个能用的任务 Agent
  • 工具与框架：选择合适的运行时、执行器与抽象
  • 规划与记忆：任务分解、状态机、短长记忆与检索
  • Agent + RAG：让 Agent “会找资料、会用资料”
  • 多智能体协作：分工、协议与社会性结构
  • 评测与安全：可度量的改进与可靠性保障
  • 案例与实验：行业模板与动手实操
