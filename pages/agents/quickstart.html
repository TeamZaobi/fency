目标：实现一个“资料检索 + 结构化总结”的单任务 Agent，具备基本可观测性与可复现性。

1. 问题与验收定义
  • 输入：主题关键词与限制条件（来源、时间、字段）
  • 输出：JSON Lines 每行一条事实，含 source、quote、summary、tags
  • 验收：字段齐全、来源可追溯、重复率低于阈值
2. 能力与工具
  • LLM：推理与抽取（可替换为任何支持的提供商）
  • 检索：Web 搜索 / 本地索引
  • 结构化：模式（schema）约束与校验
  • 观测：日志、失败重试、可复现随机种子
3. 骨架示例（伪代码）
// plan
const plan = ["search(kw)", "select(topN)", "extract(schema)", "dedup()", "export(jsonl)"];

// tool binding
const tools = { search, fetch, parse, validate, exportJSONL };

// loop
for (const step of plan) {
  try { await run(step, tools, ctx); } catch (e) { log(e); retry(step); }
}

// eval
assert(metrics.coverage >= 0.8);
assert(metrics.dup_rate <= 0.1);
提示：先把“可验证的产物”做出来，再逐步把鲁棒性与通用性补齐。
